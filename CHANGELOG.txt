# Neue Spawn Methode: T8_UnitsZone

Erstellt eine Zone um den angegeben Marker und Spawn Gruppen sobald Einheiten der eingestllten Fraktion in der Nähe sind.
Verlassen die Einheiten die Zone werden die Einheiten despawnt, bereits gefallene Einheiten werden berücksichtigt beim nächsten spawn.

BSP:	[ "ArraySpawnZone", "MarkerSpawnZone", "WEST" ] spawn T8_UnitsZone;

[
	_unitsArray,	- vordefinierter Array der zu spawnenden Einheiten
	_marker,	- Marker als Zentrum der Zone
	_actSide,	- Seite die die Zone aktiviert
	_distance,	- (optional) entfernung ab wann die Zone aktiviert wird
	_condition,	- (optional) Trigger Bedienungen standart: "this"
	_onAct,		- (optional) Scripte die bei Aktivierung ausgeführt werden sollen
	_onDeAct	- (optional) Scripte die bei De-aktivierung ausgeführt werden sollen
] call T8_UnitsZone

[

		ARRAY, 		- Array mit Einheiten die gespawnt werden

	],
	
	[ 
		BOOL,		- (optional) Einheit kann um Hilfe rufen ( true )
		BOOL		- (optional) Einheit kann zur Hilfe kommen ( true )
	]					
]

Beispiele:

// Spawnt eine Gruppe am Marker "MarkerRED", als reine Infantrie, der standard Seite, führt Patrol aufgabe aus, Kommuniziert und kann Helfen
[ [ <UNIT ARRAY HERE>, "MarkerRED" ], [ "PATROL" ] ],

// Spawnt eine Gruppe am Marker "MarkerGUR", als reine Infantrie, der Seite Independent, führt Patrol aufgabe aus, Kommuniziert und kann Helfen, reine Inf. Gruppe
[ [ <UNIT ARRAY HERE>, "MarkerGUR", true, RESISTANCE ], [ "PATROL" ] ]

// Spawnt ein Sniperteam am "SniperSpawn" Marker mit der Overwatch Aufgabe und bewacht Marker "overwatchTHIS", Kommuniziert und kann Helfen, spawn Seite = standard, reine Inf. Gruppe
[ [ [ "O_sniper_F", "O_spotter_F" ], "SniperSpawn" ], [ "OVERWATCH", "overwatchTHIS" ] ]
	
// Spawnt einen Schützenpanzer am "vp" Marker, keine reine Inf Gruppe, Seite: EAST, Aufgabe PATROL_MARKER, mit den Wegpunkten vp1, vp2, vp3 ohne SAD Wegpunkt, kann nicht Kommunizieren und kann nicht Helfen
[ [ [ "O_APC_Tracked_02_cannon_F" ], "vp", false, EAST ], [ "PATROL_MARKER", [ "vp1", "vp2", "vp3" ], false ], [ false, false ] ],

##### V005 - 24.10.2013 #####

[add][task: OVERWATCH]:	Aufgabe bei der Sich die Gruppe eine Erhöhte Position sucht von der sie Einsicht auf 
			eine gesetzte Position (Marker) hat. Wird keine Position gefunden, die mindestens 20m erhöht ist,
			Sichtline hat, und zwischen 250m und 550m entfernt ist, führt die Gruppe einen SAD wegpunkt auf
			der gesetzten Position aus.	
			Infantrie Gruppen werden automatisch auf combatmode "RED" gesetzt geraten sie in einen Kampf, dies
			wird nach 120 sek. auf "GREEN" gesetzt, wenn sie danach wieder aus dem Kampf gelöst sind (Verhalten
			des Wegpunktes "AWARE") setzten sie sich auf "WHITE" und bekommen wieder Order auf die Position zu
			schauen. (läuft in einer "endlos" schleife es sei den die einheiten verlassen die Gruppe oder sterben.)
		 		
Aufbau des Gruppen Arrays für OVERWATCH

[ 
	ARRAY, 		- Array mit Einheiten die gespawnt werden ( array )
	MARKER-NAME,	- Marker, an dem Einheiten Spawnen und der ihr Einsatzgebiet ist ( string )
	AUFGABE, 	- Aufgabe der gespawnten Gruppe ( string )
	[ 
		BOOL,	- Einheit kann um Hilfe rufen ( bool )
		BOOL	- Einheit kann zur Hilfe kommen ( bool )
	], 		
	BOOL, 		- Einheit ist eine reine Infantrie Gruppe ( bool )
	MARKER-NAME	- ( optional : für Aufgabe OVERWATCH ) Marker, als Positon die überwacht wird, ist dieser Marker
			nicht gesetzt, wird die Spawn position überwacht!
]

Zugehörige Datei: AT8_taskOverwatch.sqf
